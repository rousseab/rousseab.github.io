This method drives the execution of an experiment. It is responsible for training a model based on a given dataset, evaluate the trained model and log all relevant information. input:

config: dictionary containing all the needed parameters dataset: the full instantiated dataset, which will be split within this driver model_class: class for the model, which should derive from ModelBase. logger_class: class responsible for keeping track of model metrics
